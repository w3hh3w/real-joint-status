# 睿尔曼机械臂状态参数面板

一个基于 Web 的睿尔曼机械臂实时状态监控与数据记录系统，支持实时数据展示、图表可视化和数据导出功能。

## 功能特性

- **实时数据监控**：实时显示机械臂各关节的速度、加速度、温度、电流和扭矩数据
- **实时图表**：动态展示关节参数变化趋势，支持选择不同参数和关节
- **数据记录**：支持开始/停止记录，自动保存关节数据
- **数据导出**：支持导出为 CSV 和 Excel 格式
- **扭矩配置**：可自定义各关节的扭矩常数

## 快速开始

### 运行程序

双击运行 `web_server.exe`，程序会自动启动 Web 服务器并在浏览器中打开控制面板页面（http://127.0.0.1:5000）

### 连接机械臂

1. 确保本机与机械臂在同一局域网内
2. 填写机械臂 IP 地址（默认：192.168.1.28）
3. 点击"自动获取"按钮获取本机 IP 地址
4. 确认端口配置正确（TCP: 8080, UDP: 8089）
5. 点击"连接机械臂"按钮

### 数据监控

连接成功后，界面会实时显示各关节的参数数据：
- 速度 (°/s)
- 加速度 (°/s²)
- 温度 (℃)
- 电机电流 (mA)
- 扭矩 (Nm)

### 数据记录

1. 连接机械臂后，点击"开始记录"按钮
2. 记录过程中会显示已记录的数据条数
3. 点击"停止记录"结束当前记录会话
4. 记录会自动保存，可在记录列表中查看

### 数据导出

在记录列表中，每条记录支持两种导出格式：
- **导出 CSV**：导出为逗号分隔的文本文件
- **导出 Excel**：导出为 Excel 表格文件

导出的文件保存在程序所在目录的 `recordings` 文件夹中。

### 实时图表

- **参数选择**：选择要显示的参数类型（速度/加速度/温度/电流/扭矩）
- **关节选择**：选择显示全部关节或单个关节
- 图表会实时更新显示数据变化趋势

### 扭矩配置

点击"⚙️ 关节参数"按钮，可以配置各关节的扭矩常数（Nm/A），用于计算关节扭矩值。

## 注意事项

- 确保防火墙允许程序访问网络
- 数据周期默认为 20ms（50Hz），可根据需要调整
- 断开连接后，记录按钮会被禁用
